## Configuración de MCP
1. **Instalación:** Se accedió a `Settings > Features > MCP` en Cursor.
2. **Servidor:** Se configuró el servidor de `filesystem` para permitir el acceso al proyecto.
3. **Uso:** Se realizaron consultas para que la IA analizara la relación entre `index.html` y `app.js` de forma simultánea. Es útil en proyectos reales para mantener la coherencia entre el backend y el frontend sin copiar código manualmente.
