## Uso de Cursor
# 1. Herramientas utilizadas
Chat de IA: he usado para preguntar dudas y corregir errores en el código.

Edición rápida: he han aplicado cambios automáticos para mejorar funciones como el guardado de tareas.

# 2. Comandos principales
Aceptar cambios: Uso de la tecla Enter para confirmar las mejoras de la IA.

Descartar: Uso de la tecla de borrar para cancelar sugerencias que no servían.

Chat lateral: Espacio donde se han pedido explicaciones sobre el funcionamiento del proyecto.

# 3. Configuración extra
Se ha activado el sistema MCP para que la IA entienda mejor la relación entre los archivos del proyecto y pueda dar respuestas más precisas.

# 4. Solución de problemas
Cuando han aparecido colores o errores de formato en el código, se han solucionado aceptando los cambios de la IA o borrando manualmente los trozos repetidos para dejar el archivo limpio.
