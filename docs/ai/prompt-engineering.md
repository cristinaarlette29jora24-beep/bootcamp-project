# Prompt Engineering

1. **Rol Senior:** "Actúa como desarrollador senior y optimiza este código". (Mejora la calidad).
2. **Restricción:** "Hazlo sin usar librerías externas". (Código más ligero).
3. **Paso a paso:** "Explica la lógica antes de escribir el código". (Evita errores lógicos).
4. **Validación:** "Añade seguridad contra ataques básicos en este input". (Código más robusto).
5. **Formato:** "Devuelve solo el código en formato JSON". (Facilita la integración).
6. **Debug:** "Encuentra el error de referencia en este bloque". (Ahorra tiempo de búsqueda).
7. **Estilo:** "Usa nombres de variables descriptivos en inglés". (Sigue estándares).
8. **Simplificación:** "Haz que esta función sea de una sola línea". (Código más limpio).
9. **Documentación:** "Genera comentarios JSDoc para esta función". (Mejor mantenimiento).
10. **Traducción:** "Explica qué hace este código en lenguaje sencillo". (Mejora la comprensión).
